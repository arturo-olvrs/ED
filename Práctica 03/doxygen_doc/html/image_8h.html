◆ LoadResult

enum LoadResult : unsigned char

Enumeración que representa el resultado de cargar una imagen PGM.

Esta enumeración se utiliza para indicar el resultado de la operación de carga de una imagen PGM. Puede tener uno de los siguientes valores:

  • LoadResult::SUCCESS: La operación de carga se realizó con éxito.
  • LoadResult::NOT_PGM: El archivo no es un archivo PGM válido.
  • LoadResult::READING_ERROR: Se produjo un error al leer el archivo.
